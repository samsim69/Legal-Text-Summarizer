Legal-Text-Summarizer
Legal Text Summarization is an end-to-end NLP-based web application designed to automatically summarize lengthy legal verdicts into concise, readable summaries. Built using Python, Flask, and the PEGASUS transformer model, the system significantly reduces the time required for legal document analysis.

Key Features
Summarizes 100+ page legal judgments into concise summaries readable in under 4 minutes
Uses PEGASUS, fine-tuned specifically on Indian legal documents for domain accuracy
User-friendly web interface built with HTML & CSS
Backend powered by Flask for seamless model integration
Developed collaboratively by a team of 3+ members
Tech Stack

Programming Language: Python
Backend Framework: Flask
NLP Model: PEGASUS (Transformer-based summarization)
Frontend: HTML, CSS
Domain: Legal NLP (Indian Legal Texts)
